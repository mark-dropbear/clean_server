# clean_server

A Dart-based server application built with the `shelf` framework, following **Clean Architecture** principles.

## Features
- **Task Management**: RESTful API for tasks and task lists.
- **Contact Us**: End-to-end feedback submission flow with a Lit-based frontend and Clean Architecture backend.
- **Modern Frontend**: SSR via Mustache templates integrated with Lit components and native ESM (JSPM).
- **JSON-LD**: Built-in support for Schema.org semantic metadata.
- **Robustness**: Comprehensive test coverage across all layers.
